## 👁️ Vision
LUXOR9 is not a dashboard; it is a **Sovereign Operating System** for AI Agent Swarms. It moves beyond static monitoring into **God-Mode Steering**, allowing a human operator to visualize, intercept, and direct a hierarchical swarm of 179+ agents in a high-fidelity spatial environment.

---

## 🛠️ Core Architecture

### 1. The Sovereign Omni-Workspace (Frontend)
A spatial-first interface designed for cognitive load reduction and absolute control.
- **Infinite Canvas Engine**: A coordinate-aware viewport with spring physics for seamless navigation of the agent constellation.
- **Sovereign Window Base**: VNC-style agent windows featuring real-time operational logs and tiered color-coding.
- **Empire Prompt (God-Mode Console)**: A command-line interface for direct neural intercept and swarm orchestration.
- **Spatial State Store**: Powered by Zustand, managing real-time X, Y, Z coordinates and operational states for the entire swarm.

### 2. Hierarchical Agent System (Backend)
A 5-tier intelligence hierarchy ensuring precision and scalability.
- **T0: LUXOR-PRIME** $\rightarrow$ The Singular Intelligence.
- **T1: C-Suite** $\rightarrow$ Strategic Directives (CORTEX, TREASURY, SENTINEL).
- **T2: VP Agents** $\rightarrow$ Operational Oversight.
- **T3: Managers** $\rightarrow$ Execution Coordination (ACQ/DEL/RET).
- **T4: Workers** $\rightarrow$ Atomic Task Execution.

## ⚡ God-Mode Commands
The **Empire Prompt** allows for instantaneous steering of the swarm:

| Command | Effect | Visual Feedback |
| :--- | :--- | :--- |
| `/intercept [id]` | Forces an agent into an intercepted state | Yellow Pulse + Neural Lock Badge |
| `/resume [id]` | Releases the agent back to the swarm | Return to standard operational color |
| `/broadcast [msg]` | Pushes a sovereign directive to all agents | Synchronized log updates across all windows |
| `/halt` | Immediate suspension of all swarm activity | System-wide state transition to `idle` |
| `/snap [id]` | Instant viewport translation to agent coordinates | Smooth spring-motion camera snap |

## 🚀 Quick Start

### Backend Deployment
```bash
cd backend
pip install -r requirements.txt
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

### Frontend Launch
```bash
cd frontend/luxor9-final
npm install
npm run dev
```
